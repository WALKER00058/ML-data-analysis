# 📊 AI Data Insights Tool

An AI-powered Streamlit app that analyzes datasets, fills missing values, and predicts data using machine learning.

---

##  Live App

 [Use the app here]([https://your-app.streamlit.app](https://ml-data-analysis-35vfycdxupjhgrvdu8jhnd.streamlit.app))

---

##  Run Locally

1. Clone the repo:

```bash
git clone https://github.com/WALKER00058/ML-data-analysis.git
cd ML-data-analysis
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```

---

##  Features

* Fill missing data using ML
* Predict unknown fields
* Detect anomalies
* Correlation insights
* Model performance metrics

---

##  Input

Upload any CSV file with numeric and categorical data.

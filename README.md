# 📊 AI Data Insights Tool

AI Data Insights Tool is a web app built with **Streamlit** and **Scikit-learn** that lets you analyze any CSV dataset quickly. The app automatically detects anomalies, fills missing values, predicts unknown fields based on partial input, and provides model performance insights including accuracy, precision, recall, and feature importance.  

This tool is perfect for data enthusiasts, students, and anyone who wants instant AI-powered insights on their dataset.

---

## 🚀 Features

- **Basic Dataset Info**: Quickly see rows, columns, and missing value counts.  
- **Correlation Insights**: Detect strong relationships between numeric columns.  
- **Anomaly Detection**: Identify unusual or outlier data points.  
- **Fill Missing Values**: Automatically fill missing values using Random Forest models.  
- **Predict Missing Fields**: Enter n-1 fields and let the app predict the remaining field.  
- **Model Insights**: Train predictive models, get accuracy, precision, recall, F1 score, and feature importance.  
- **Download Cleaned Data**: Export the dataset after filling missing values.
